# Simple Python Implementation of Conway's Game of Life

## Quick start 

1. Clone the repo
   ```sh
   git clone https://github.com/jenkoj/game-of-life
   ```
2. Run the script
   ```sh
   python3 game.py
   ```
3. Output should be seen in terminal


Output is also saved to GameOut.png
